Play online: [StephenBarnes.github.io/ANS-trainer](https://stephenbarnes.github.io/ANS-trainer/)

This is a tool for training estimation of numbers using a cognitive system called the [approximate number system](https://en.wikipedia.org/wiki/Approximate_number_system). Each round shows a field of symbols. Type your estimated number and press Enter to submit.

Implemented entirely by Codex (GPT-5.4 medium).
